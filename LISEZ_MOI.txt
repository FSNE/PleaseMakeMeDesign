Projet
======

"Please Make Me Design" est à l'origine le matériel nécessaire au "Screen Party" d'Open Source Publishing ayant lieu le 22 avril 2010 à l'ESAPV Mons.
"Please Make Me Design" est une initiation à la ligne de commande par le biais du design graphique, de l'écriture et de la poésie concrète.
Ce pack collecte textes et didacticiels de références, ainsi qu'une collection de scripts. Il a été créé pour faciliter la collaboration, mais aussi pour pouvoir être augmenté par la suite pour d'ultérieurs workshops.


Pour commencer
==============

- Placer ce dossier dans votre dossier de travail
- Ouvrir une fenêtre de Terminal
  - sur MacOS :
    - Aller dans Applications/Utilitaires/Terminal
- Aller dans votre dossier de travail :
  $ cd /Volumes/votre-disque-dur/.../votre-dossier-de-travail/
  (la commande *cd* permet de changer de répertoire, en anglais *Change Directory*)
    - *ASTUCE* la touche TAB vous permet d'auto-compléter les noms de dossier et de fichier; ainsi vous n'avez pas besoin de connaître l'arborescence du disque dur et les noms de fichiers par cœur!
- Aller dans le dossier script du pack "PleaseMakeMeDesign" :
  $ cd pleasemakemedesign/scripts
  
Solution alternative
--------------------
- Placer ce dossier dans votre dossier de travail
- Ouvrir une fenêtre de Terminal
  - sur MacOS :
    - Aller dans Applications/Utilitaires/Terminal
- taper "cd"
- Trouver le dossier du pack dans le Finder
- Glisser/déposer ce dossier dans le terminal
- Appuyer sur Entrée


Astuces
=======

- "../" signifie dossier supérieur
- taper "-h" ou "--help" après le nom d'un programme permet d'afficher l'aide 


Expérimenter
============

Vous pouvez à présent tester les lignes de commandes décrites dans les fichiers references/techniques/coreutils.txt ou references/techniques/exemples.txt.
N'ayez pas peur de casser votre ordinateur, à part les commandes *rm* et *>*, rien ne modifie vos fichiers. *rm* supprime un fichier (à utiliser avec précaution car le fichier est directement supprimé sans passer par la corbeille!). La commande *>* permet d'enregistrer le résultat obtenu dans un fichier.

- Pour appeler un fichier se situant dans un sous-dossier, taper :
  $ mon-dossier/mon-fichier.txt
- Pour appeler un fichier dans un dossier supérieur, taper :
  $ ../mon-fichier.txt


Usage
=====

twitter.py
----------
Recherche un mot sur Twitter.

  $ python twitter.py -r "bonjour"

- options (à taper après l'appel du programme. "-r" est le raccourci de "--requete")
  -r, --requete : mot ou groupe de mots à recherche
  -l, --langue  : choix de langue (2 lettres, par ex. "fr", "en", "all" pour n'importe quelle langue)
  -t, --tag     : cherche un thème
  -n, --nombre  : nombre de résultat souhaité
  -d, --debut   : date de début de recherche (format aaaa-mm-jj)
  -f, --fin     : date de fin de recherche (format aaaa-mm-jj)


make_svg.py
-----------
Convertit le fichier .txt en fichier .svg afin d'être modifié dans Inkscape

  $ echo "bonjour" | python make_svg.py
  $ cat mon_fichier.txt | python make_svg.py


filtre_mot.py
-------------
Cherche un mot (ou groupe de mots) dans chaque phrase d'un texte et place les phrases de manière à ce que le mot sélectionné soit toujours au milieu.

  $ echo "bonjour" | python filtre_mot.py --mots "bonjour"
  $ cat mon_fichier.txt | python filtre_mot.py -m "bonjour"


frequence_mots.py / frequence_mots_bloc.py
------------------------------------------
À partir d'un lexique, change la taille d'un mot en fonction de son nombre d'occurrence dans un texte.
La version "bloc" place tous les mots dans un même paragraphe.

  $ cat mon_lexique.txt | python frequence_mots.py
  $ cat mon_lexique.txt | python frequence_mots_bloc.py
  

Affiner sa recherche Twitter
============================

- utiliser des guillemets pour chercher un groupe de mots exacts: "ma recherche"
- ajouter "?" pour ne rechercher que des questions
- ajouter "-rt" pour ne pas recevoir les re-tweets
- ajouter "filter:links" pour ne recevoir que des liens
- ajouter "#" devant un mot pour chercher un thème
- utiliser des smileys pour chercher une humeur: ":)", ":("
- utiliser le signe moins '-' pour omettre un terme de la recherche
- utiliser "since:aaaa-mm-jj" et "until:aaaa-mm-jj" pour chercher des tweets sur une période précise
- "from:utilisateurX": message écrit par l'utilisateurX
- "to:utilisateurX": message destiné à l'utilisateurX
- "@utilisateurX": message concernant l'utilisateurX
- ajouter "near:90210" (code postal Américain) ou "near:Belgium" pour ne recevoir que des tweets locaux
